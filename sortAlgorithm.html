Sorting Algorithm Visualizer

Github Repository

Written in Java and GUI with JavaFX. This was my own summer project.

What is it?

This was my first big personal project, to visualize the algorithms that I have been learning. It sorts all the bars according to their height using the Insertion Algorithm.

How does it work?

It has 3 different classes, AlgorithmTest, the driver class that starts the JFrame and runs the program. Then the SortingPanel which extends the JPanel class. This class contains the array, the colors of the bars and does the proper visualization. Lastly is the Insertion class, which starts a thread, that sleeps each time two bars are being compared, so the algorithm is visible and not instantly. It also takes the whole Panel, so I can utilize the highlightSwap() method in each comparison within the algorithm.
This was a huge learning challenge for me, since I didn't know much JavaFX before this. My next challenge will be to make a web app with React that makes this and many more sorting algorithms!
