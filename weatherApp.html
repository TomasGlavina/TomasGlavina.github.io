Weather App

Github Repository

Written in HTML, CSS, JavaScript and charts with Chart.js. This web app was the final assignment for our Web Development course at TAMK.

What is it?

It is a web application designed to show weather data gathered by the school's weather station and provided by an API also from TAMK. It has various different types of data, such as wind speed, temperature, rain, etc. It also has different time spans, like last 24 hours, 3 days, etc.

How does it work?

It's a single HTML page that it updates the different views according to the data requested by the user with JavaScript and FetchAPI. I created a dropdown menu that lets you select the different time intervals. With chartJS I grab the data and put it into a bar or line chart.
UI is simple and easy to use, trying to be as clean as possible but keeping the options complex.
